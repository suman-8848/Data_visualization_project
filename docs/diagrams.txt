DATA FLOW DIAGRAM
=================

User Question
     ↓
[LLM Inference] → Answer + Attention Weights
     ↓
[Entity Extraction] → Entities (PERSON, GPE, ORG, etc.)
     ↓
[Relationship Extraction] → Relations (subject-verb-object)
     ↓
[Graph Construction] → Nodes + Edges + Layout
     ↓
[JSON Response] → {answer, tokens, attention, graph}
     ↓
[D3.js Rendering] → Interactive Visualizations
     ↓
[User Interaction] → Synchronized Highlighting
